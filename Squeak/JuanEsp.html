PhotoSqueak - Procesamiento de Imágenes

    Esta es la implementación de los trabajos prácticos que hice para la materia "Introducción al Procesamiento de Imágenes" en la Facultad de Ciencias Exactas y Naturales de la Universidad de Buenos Aires, durante el 1er cuatrimestre de 2000.

Algunos de los aspectos interesantes de este trabajo incluyen:

  • Implementación de un conjunto importante de rutinas para Procesamiento de Imágenes
  • Uso de Squeak Smalltalk, una herramienta de Objetos
  • Construcción de una interfaz al usuario cómoda y vistosa en un framework novedoso (Morphic)
  • Uso de un recurso de generación automática de código 'C' para crear dlls para las rutinas más pesadas
